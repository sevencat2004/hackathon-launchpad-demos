# Hackathon Launchpad

Four parallel hackathon tracks started on 2026-06-04.

This workspace is the first-stage control room for:

- H0: Hack the Zero Stack
- UiPath AgentHack
- Sui Overflow 2026
- Splunk Agentic Ops Challenge

The shared strategy is to reuse one strong theme where possible:

> Agentic bounty and operations intelligence: discover tasks, rank payout probability,
> coordinate actions, explain risks, and produce auditable handoff packages.

Each track has a separate project folder under `projects/`. First-stage completion means:

- official source and deadline captured
- product concept selected
- MVP scope defined
- architecture sketched
- submission assets listed
- user-owned account/payment steps separated
- local preflight passes

## Current Ranking

1. `h0-zero-stack` - first build target.
2. `uipath-agenthack` - automation-focused build target.
3. `splunk-agentic-ops` - narrow quick-build target due to short deadline.
4. `sui-overflow` - medium-term ecosystem target.

## User-Owned Actions

Codex can build project assets, code, docs, and demo materials. The user must personally handle:

- Devpost registration and final submission clicks
- platform accounts and login verification
- Vercel, AWS, UiPath, Splunk, or Sui wallet authorization
- tax, KYC, identity, and payment onboarding
- any private key, seed phrase, bank, tax ID, or one-time-code handling
