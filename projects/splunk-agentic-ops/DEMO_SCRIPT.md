## Video Length

Target: 90 seconds.

## Script

1. Show synthetic checkout logs.
2. Run the incident analyzer.
3. Highlight error count, slow events, and max latency.
4. Show probable cause.
5. Show generated runbook steps.
6. Show postmortem draft.
7. Explain how the same flow can connect to Splunk search/API after account access.

## Voiceover Draft

Runbook Relay Agent turns raw service events into an incident handoff. In this demo,
payment provider timeouts trigger retries and circuit breaker activation. The agent
summarizes severity, signals, probable cause, runbook steps, and a postmortem draft.
