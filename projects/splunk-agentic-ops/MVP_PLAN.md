# MVP Plan

Status: submitted to Splunk Agentic Ops Hackathon, not approved, not paid.
Price: 20,000 USD total cash and feedback awards shown by official Devpost rules.

## MVP Slice

Build a demo that:

1. Loads synthetic service logs.
2. Detects anomaly patterns.
3. Explains incident likely cause.
4. Selects a runbook.
5. Generates an operator handoff and postmortem.

## Technical Shape

- Local app with synthetic logs first.
- Splunk search/API integration after account access is clear.
- Demo can show both raw data and agentic explanation.

## First Code Tasks

- Create synthetic log fixture.
- Create incident scoring/explanation function.
- Create runbook mapping.
- Prepare Splunk integration checklist.
