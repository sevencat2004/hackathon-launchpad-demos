## Candidate Product

Working title: Runbook Relay Agent.

An agentic operations assistant that reads incident logs, explains probable root cause,
selects a runbook, drafts remediation steps, and produces a postmortem summary.

## Why This Can Win

It is narrow enough for the short deadline and demonstrates agentic ops value with clear
before/after output: raw logs become triage, action plan, and audit summary.

## First-Stage Decision

Proceed only as a narrow MVP. Use synthetic logs unless user provides a Splunk data source.
