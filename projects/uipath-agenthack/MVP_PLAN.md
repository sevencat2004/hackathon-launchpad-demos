# MVP Plan

Status: first-stage started, not submitted, not paid.
Price: 50,000 USD cash prize pool.

## MVP Slice

Build a UiPath-oriented workflow that:

1. Receives opportunity links or messages.
2. Extracts amount, deadline, platform, account gates, and competition signs.
3. Produces a ranked queue.
4. Generates a user approval packet.

## Technical Shape

- UiPath workflow or agent definition.
- Local deterministic scoring module.
- Sample inbox CSV/JSON fixture.
- Demo scenario using four current hackathon candidates plus existing bounty examples.

## First Code Tasks

- Create scoring and queue fixture shared with H0.
- Create UiPath implementation notes.
- Prepare demo script for Studio/Web platform once access is confirmed.
